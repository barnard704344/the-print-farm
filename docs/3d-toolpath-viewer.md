# 3D Toolpath Viewer

The File Library viewer supports interactive toolpath inspection with high detail settings.

## Capabilities

- Interactive 3D camera controls
- Feature-based colours (walls, bridges, ironing, supports, interfaces)
- Staged loading/progress feedback for large files
- High-fidelity arc interpolation and move density

## Performance Notes

Large files can take longer to parse. Current defaults prioritize detail and visual quality over raw speed.
