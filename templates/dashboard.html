Send G-code from OrcaSlicer to the Print Farm

1. Download the Upload Script

This script runs automatically after OrcaSlicer finishes slicing and sends the G-code to the farm's job queue.

⬇ Windows (.bat) ⬇ Python (Any OS)

Windows users: just download the .bat file. No PowerShell or Python needed — it uses curl which is built into Windows.

2. Setup in OrcaSlicer

1
Save the script
Download the script above and save it somewhere permanent, e.g.
Windows: Leave in your Downloads folder (default download location)
Mac/Linux: ~/upload_to_farm.py
2
Open Post-Processing Scripts
First, make sure OrcaSlicer is in Advanced or Expert mode (top-right corner dropdown — it's hidden in Simple mode).
Then click the pencil/edit icon next to your print profile name to open the Process settings.
Click the Others sub-tab at the top of the settings panel, then scroll to the bottom to find Post-processing Scripts.
3
Add the script command
Paste this into the Post-processing Scripts box:

Windows:
C:\Windows\System32\cmd.exe /c C:\Users\USERNAME\Downloads\upload_to_farm.bat
Replace USERNAME with your Windows username (e.g. C:\Windows\System32\cmd.exe /c C:\Users\rob.barnard\Downloads\upload_to_farm.bat).

Python (any OS):
python3 "/path/to/upload_to_farm.py"
Update the path to match where you saved the script.
4
Slice as normal
Click "Slice" in OrcaSlicer. After slicing completes, the script will automatically upload the G-code to the farm. You'll see a confirmation in OrcaSlicer's console log.

Notes

  • The script runs after every slice — uploaded files appear in the Job Queue tab.
  • Only .gcode files are uploaded. OrcaSlicer passes the file path automatically.
  • The script is silent on success in OrcaSlicer — check the Job Queue to confirm.
  • Make sure your computer can reach on the network.
  • You can also drag & drop files directly into the Job Queue tab as an alternative.
